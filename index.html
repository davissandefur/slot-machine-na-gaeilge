Welcome to the Irish Language Slot Machine.

Hit the spin button to begin.

After you have spun, and your words pop up, create a sentence using them. For example, if you get "is", "mé", and "dochtúir", as in the loading sample, you could say "Is dochtúir mé." or "Ní dochtúir mé" if you wanted to use the negative. This is best used in conjunction with a teacher/fluent speaker to test if you're correct.

is mé dochtúir
